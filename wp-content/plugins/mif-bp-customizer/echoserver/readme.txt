== Требуется предварительная установка:

1. node.js и socket.io (для работы сервера)
2. forever (для скрипта автозапуска)

== Установка:

1. Установить то, что требуется предварительно
2. Скопировать server.js в папку /root/bin/echoserver
3. Скопировать скрипт автозапуска в /usr/local/etc/rc.d (только FreeBSD)

== Запуск для отладки:

node server.js

== Запуск для работы:

/usr/local/etc/rc.d/echoserver start (только FreeBSD)

== Логи:

/var/log/forever.log

== Linux и др.

Всё решаемо. Запустите сервер так, как рекомендуется для вашей операционной системы

== Настройка

В файле server.js укажите адрес, порт и общий ключ вашего веб-сервера

== Безопасность

Укажите свой общий ключ веб-сервера и эхо-сервера. Не оставляйте рабочий сервер в папке 
плагина - общий ключ может узнать кто-то еще. Это потенциально дает возможность кому-то 
делать рассылки вашим пользователям.
