Плагин расширения возможностей BuddyPress для создания сайта социальной сети.

== Description ==

Плагин расширения возможностей BuddyPress для создания сайта социальной сети. 
Ориентирован на работу совместно с плагином социальной сети BuddyPress. 

Добавляет возможности:

1. Лента активности

Особая лента активности

	Меняет вид и поведение ленты активности на страницах пользователей 
	(на личной старнице - "Вся лента", на страницах других пользователей - только их 
	активность). Позволяет использовать инструменты блокировки контента.

Типы записей ленты активности

	Позволяет указывать типы активности, которые должны отображаться в ленте пользователя 
	(требуется установка опции "Особая лента активности").

Блокировка пользователей

	Позволяет вести списки пользователей, информация которых блокируется в вашей ленте 
	активности (требуется установка опции "Особая лента активности").

2. Поведение сайта

Профиль как домашняя страница

	Назначить профиль пользователя его домашней страницей.

Приватность профиля

	Разрешить пользователям ограничивать доступ к своему профилю.

Подписчики

	Включить возможность подписки на обновления пользователей 
	(подписка = односторонняя дружба).

Уведомления

	Продвинутый режим уведомлений.

Всплывающие сообщения

	Механизм всплывающих уведомлений (требуется настройка эхо-сервера).

Документы

	Создание коллекций файлов и документов на страницах пользователей и групп.
	Публикация файлов и документов в ленте активности.

Диалоги (экспериментально)

	Простые и удобные диалоги вместо стандартной системы личных сообщений 
	(экспериментально; требуется настройка эхо-сервера).

Фоновое изображение

	Разрешить использовать пользовательское изображение в качестве фона для профиля 
	пользователя или группы.

Адрес группы

	Разрешить изменять адрес группы в её настройках и при создании.

Кнопка «Нравится»

	Кнопка «Нравится» для записей ленты активности.

Кнопка «Репост»

	Повторная публикация (репост) записей ленты активности.	

Кнопки «Избранное», «Удалить»

	Особые кнопки «Избранное», «Удалить» (в стиле "Нравится" и "Репост").

3. Визуальные элементы

Виджет участников сайта

	Простой и быстрый виджет аватаров участников сайта.

Виджет групп

	Простой и быстрый виджет аватаров групп.


== Installation ==

1. Установить как и любой другой плагин.
2. Активировать для сети или для сайта, где используется плагин BuddyPress.
3. Зайти (в консоли) "Настройки" -> "BP Customizer", выбрать нужные опции.
